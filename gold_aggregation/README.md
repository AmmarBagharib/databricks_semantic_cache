# Gold Aggregation Layer: Analytics

This folder contains aggregation logic that turns classified conversations
into actionable insights for product teams.

---

## Country-Level Aggregation

Notebook: `7-1_gold_class_country.ipynb`

- Groups conversations by country and category
- Computes category distributions per region
- Identifies dominant conversation themes by geography

---

## User-Level Aggregation

Notebook: `7-2_gold_class_user.ipynb`

- Aggregates categories per synthetic user
- Calculates penetration metrics
- Helps distinguish common vs niche intents

---

## Purpose

These aggregations demonstrate how classified conversation data can be:
- Queried efficiently
- Visualized in dashboards
- Used to guide product decisions
